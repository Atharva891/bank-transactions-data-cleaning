## Bank Transactions Data Cleaning (Python - Pandas)

### Objective
Clean raw banking transaction data to make it analysis-ready.

### Tasks Performed
- Identified and handled missing values
- Standardized date formats
- Removed duplicate transactions
- Cleaned and standardized categorical values
- Converted amounts to numeric format

### Tools Used
- Python
- Pandas

### Outcome
Generated a clean dataset suitable for financial analysis.
